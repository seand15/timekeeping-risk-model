# Timekeeping Compliance Risk Model

This project builds a predictive model to identify employees at risk of future timekeeping compliance violations.

I simulated workforce timekeeping data (500 employees, 180 days) and engineered behavioral features like anomaly trends and overtime patterns.

A Random Forest model achieved ~0.81 ROC-AUC and was tuned to balance precision and recall for practical use.

Key finding: manager-level behavior was the strongest predictor of risk.
